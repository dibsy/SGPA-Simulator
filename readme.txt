Instructions for using the web app

Open the index.html file with Firefox or Chrome .

1.The first column represents the subject code.Its optional to fill this. 
2.The second column represents the Maximum credit assigned to that 
particular subject.Fill it with digits.
3.The third column represents grade. FILL IT IN UPPERCASE ALPHABETS ONLY.
4.Click SGPA.

NOTE:
It is tested in Firefox and Chrome only.
This version only includes only 7 subjects .
Error handling methods are not used.So if you enter abstract values you 
won't get any proper result.

Please feel free to report any bugs and problems as well as new features 
you would like to see.
